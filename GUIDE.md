# Servit 사용 설명서

## 접속

1. 브라우저에서 서버 주소 접속 (예: `http://서버IP:8765`)
2. 사용자 이름 + 비밀번호 입력 (Linux 시스템 계정 또는 `--pass` 옵션으로 설정한 비밀번호)
3. iPhone: Safari에서 공유(↑) 버튼 → "홈 화면에 추가" → 앱처럼 사용 가능
4. Android: Chrome 메뉴(⋮) → "홈 화면에 추가"

### Cloudflare Tunnel 사용 시

```bash
servit --tunnel
```

터널 URL 확인:

```bash
servit url
# 또는
grep -o 'https://.*trycloudflare.com' /tmp/servit-tunnel.log | tail -1
```

## 화면 구성

```
┌──────────────────────────────────┐
│ ☰  ● Terminal               ⏻  │  <- 상단 바
├──────────────────────────────────┤
│                                  │
│    터미널 화면                    │  <- xterm.js
│                                  │
├──────────────────────────────────┤
│ 한글 입력 중...                   │  <- 조합 표시 (한글 입력 시)
│ [입력창................] [▼]     │  <- 입력 바
├──────────────────────────────────┤
│ ESC TAB 중지 붙여넣기 Y N ...    │  <- 퀵바
└──────────────────────────────────┘
```

- **☰**: 파일 탐색기 (사이드바) 열기
- **● Terminal**: 현재 탭 표시
- **⏻**: 로그아웃

## 입력 바 (한글 입력)

| 동작 | 설명 |
|---|---|
| 입력 바에 타이핑 | 한글/영어 모두 가능. 한글 조합 완벽 지원 |
| Enter | 입력한 텍스트를 터미널로 전송 |
| 빈 상태에서 Enter | 엔터만 전송 (터미널에서 Enter 누른 것과 동일) |
| ▼ 버튼 | 키보드 닫기 |
| 입력 바 포커스 시 | 퀵바 자동 숨김 (화면 넓어짐) |
| 입력 바 포커스 해제 시 | 퀵바 자동 표시 |
| Ctrl+C (입력 바에서) | 터미널에 Ctrl+C 전송 |
| 붙여넣기 | 입력 바에 길게 눌러 붙여넣기 후 Enter |

> 영어 명령어는 터미널을 직접 탭해서도 입력 가능 (xterm.js 기본 입력). 한글은 반드시 하단 입력 바를 사용하세요.

## 파일 뷰어

| 파일 타입 | 표시 방식 |
|---|---|
| 코드 (.py, .js, .kt, .java 등) | 문법 강조 + 라인 넘버 |
| 텍스트 (.txt, .md, .log 등) | 일반 텍스트 + 라인 넘버 |
| 엑셀 (.xlsx, .xls) | 시트 탭 + HTML 테이블 |
| HWP (.hwp) | 텍스트 추출 (서식 제외) |
| DOCX (.docx) | 제목/본문/표 구조 |
| 바이너리 (이미지, 압축 등) | 파일 타입 + 크기 표시 |

- X 버튼으로 뷰어 닫기 -> 터미널로 복귀
- 500KB 이상 파일은 자동 생략
- 5000줄 이상은 잘림 표시
- **편집**: 편집 버튼 클릭 -> 내용 수정 -> 저장 버튼

## 서버 모니터링

상단 바에서 Monitor 탭 선택 시 표시됩니다.

- **CPU**: 사용률 + 코어 수
- **RAM**: 사용량 / 전체 용량
- **Disk**: 사용량 / 전체 용량
- **GPU** (NVIDIA 전용): 사용률, VRAM, 온도, 전력
- **Network**: 업로드/다운로드 속도
- **Uptime**: 서버 가동 시간
- **프로세스**: CPU/메모리 순 정렬, Kill 기능

## Pane 선택기 (tmux 전용)

tmux에서 여러 pane이 열려 있을 때, 모바일에서 특정 pane만 확대해서 보는 기능.

1. 퀵바 **Pane** 버튼 탭
2. 번호(0~8) 탭 -> 해당 pane 선택 + 전체화면 확대
3. **원래대로** -> 확대 해제 (원래 레이아웃)
4. **닫기** -> 확대 해제 + 선택기 닫기

> pane 번호는 tmux가 부여한 번호. `Ctrl-b q`로 확인 가능.

## tmux 터치 스크롤

tmux 안에서는 기본적으로 손가락 스크롤이 안 됩니다.

1. **더보기** -> tmux -> **터치** 버튼 탭 (한 번만)
2. 이후 손가락으로 위아래 스와이프하면 스크롤 됨
3. 세션마다 한 번 설정하면 유지됨

## 서버 관리 명령어

```bash
# 서비스 시작 (백그라운드 + 터널)
servit start --tunnel

# 서비스 상태 확인
servit status

# 터널 URL 확인
servit url

# 서비스 중지
servit stop

# 포그라운드 실행 (디버깅)
servit --port 8765

# systemd 사용 시
systemctl --user status servit servit-tunnel
systemctl --user restart servit
journalctl --user -u servit -f
```

## 보안

| 항목 | 설명 |
|---|---|
| 인증 | Linux 시스템 비밀번호 (su 명령 검증) |
| brute force 방어 | 5분 내 10회 실패 -> 10분 잠금 |
| 통신 암호화 | HTTPS (Cloudflare Tunnel 사용 시) |
| 쿠키 | HttpOnly, SameSite=Strict (JS 접근 불가) |
| 세션 | 토큰 기반, 서버 재시작 시 만료 |
| 로그아웃 | 상단 바 ⏻ 버튼 |

> 외부 접속 시 반드시 `--tunnel` 옵션으로 HTTPS를 사용하세요.

## 알려진 제한

- tmux 터치 스크롤은 세션마다 수동 활성화 필요
- Cloudflare 무료 터널은 재시작 시 URL 변경됨
- HWP 파일은 텍스트만 추출 (이미지/서식 미지원)
- 500KB 이상 파일은 뷰어에서 열 수 없음
- xterm.js에서 직접 한글 입력 시 자모 분리됨 -> 반드시 하단 입력 바 사용
- 로그 파일 읽기는 현재 사용자 권한에 따름 (root 로그는 권한 필요)

## 문제 해결

| 증상 | 해결 |
|---|---|
| 접속이 안 됨 | `servit status`로 실행 상태 확인. 방화벽에서 포트 열기 |
| 한글이 깨짐 | xterm.js 직접 타이핑 대신 하단 입력 바 사용 |
| GPU 모니터링 안 됨 | `nvidia-smi` 명령이 실행 가능한지 확인 |
| Excel 파일 안 열림 | `pip install openpyxl` |
| HWP 파일 안 열림 | `pip install pyhwp` |
| Docker 목록 비어 있음 | Docker 설치 여부 및 현재 사용자 docker 그룹 확인 |
| 터널 URL 변경됨 | Cloudflare 무료 터널 특성. `servit url`로 새 URL 확인 |
